Style Guide

Paleta, tipografia e componentes extraídos da marca Sangue Doce. Referência canônica para desenvolvimento de produto e comunicação.

Nota de alinhamento: A logo usa navy + laranja como identidade primária, enquanto a implementação atual do site adota verde (#3f7a4f) e papel quente (#faf6ee). Este guia documenta as cores da logo como referência canônica. Alinhar o site a esta paleta é um passo estratégico que vale discutir.

Uso

Diretrizes de Aplicação

Fazer
  • Usar navy como fundo de header e hero — ele ancora a identidade da logo
  • Reservar Energy (laranja) para o único CTA de maior hierarquia na tela
  • Usar Spark (ciano) para estados positivos, confirmações e hover
  • Combinar Navy + Azure + White para seções informativas neutras
  • Manter texto sobre Energy em branco puro, sem meios-tons
  • Usar a escala tonal de Navy para superfícies e fundos de seção
Não fazer
  • Usar Energy como cor de fundo de seções inteiras — é um acento, não uma base
  • Misturar a paleta verde do site (#3f7a4f) com Navy sem transição de contexto
  • Colocar texto em Azure sobre fundo branco em tamanho menor que 18px
  • Usar Spark como cor de texto em fundo claro — contraste insuficiente
  • Combinar Navy escuro com Energy em texto corrido — 3.4:1, abaixo de AA
  • Usar gradiente de Navy → Energy como fundo de card ou hero decorativo
